RapidGame upgrade instructions:

* Check in existing edits to RapidGamePro and RapidGame.
* Move old cocos2d-js folder out.
* Copy new cocos2d-js folder in.
* Remove samples and other extraneous folders.
* Git add --all and commit.
* Apply existing cocos2d.patch manually to new cocos2d-js folder.
* Check in.
* Upgrade package.json version and cocos2djs URL in rapidgamepro.js AND src/downloaded.txt.
* Check in.
* Verify rapidgamepro prebuild (sudo npm unlink rapidgamepro -g; cd ~/code/RapidGamePro; sudo npm link .).
* Verify TwoScene iOS, Mac, Android.
* Check in any tweaks.
* Build new patch file (see bottom of rapidgamepro.js).
* Check in new patch file.
* make rg && cd ../releases/RapidGame && sudo npm uninstall rapidgamepro -g; sudo npm uninstall rapidgame -g; sudo npm install . -g && rapidgame prebuild
* Test on Windows
* Update Changelog.
* Check in any final edits to RapidGamePro and make rg.
* Commit and push new RapidGame, create release on Github.
* Publish RapidGame using sudo npm publish .
* Update docs (na rez/docco/docco.jst && scp -r ../releases/RapidGame/docs/* ssl.wizardfu.com:w/wizardfu.com/docs/rapidgame/)
* Update SpriteDraggerX
